# Neuro-Persuasion Toolkit

Framework para aplicar neurociencia del comportamiento y psicología del consumidor a tres dominios: **SEO/SEM/MEO**, **copywriting/marketing** y **ventas (frías, consultivas, cierre)**. Los tres dominios comparten el mismo modelo de circuitos cerebrales pero lo aplican a piezas distintas (una meta description no es un cold call). Este SKILL.md es el router: identifica el dominio, carga solo la(s) referencia(s) relevante(s), y siempre aplica las reglas de `calibracion-y-etica.md`.

## Antes de escribir cualquier copy: lee `references/calibracion-y-etica.md`

Este archivo es obligatorio, no opcional, cada vez que este skill se activa. Contiene dos cosas críticas:
1. Qué afirmaciones del framework son neurociencia sólida vs. heurística popular de marketing (para no presentarle a un cliente una cifra de blog como si fuera un paper).
2. Guardrails éticos de persuasión no engañosa — el usuario tiene un compromiso explícito con copy honesto, especialmente en funnels de WhatsApp.

## El modelo de circuitos (marco compartido)

| Circuito | Qué procesa | Dónde aparece en el copy |
|---|---|---|
| Amígdala | Amenaza, miedo, urgencia, pérdida | Titles, headlines, subject lines, apertura de cold call |
| Núcleo Accumbens (NAc) | Anticipación de recompensa, deseo | CTAs, beneficios aspiracionales, descuentos, gamificación |
| Ínsula | "Dolor de pagar", fricción, riesgo | Pricing, garantías, trial sin tarjeta, manejo de objeción de precio |
| Corteza Prefrontal (CPF) | Racionalización, comparación, justificación ante terceros | ROI calculado, comparativas, features, datos concretos |
| Neuronas espejo | Empatía, identificación, prueba social | Testimonios, video, storytelling, casos de uso |
| Ganglios basales | Hábito, rutina, automatización de conducta | Onboarding, digests, notificaciones de logro, retención |
| Corteza cingulada anterior | Curiosidad, detección de novedad/conflicto | Apertura de conversación, pattern interrupts, loops abiertos |
| Sistema oxitocinérgico | Confianza, vínculo social | Discovery consultivo, rapport, venta relacional |

Secuencia neurológica típica en cualquier pieza persuasiva: **amígdala → NAc → ínsula → CPF**, con neuronas espejo y ganglios basales actuando de forma transversal. Regla general: activar primero el circuito emocional (amígdala/NAc/espejo) y usar el racional (CPF) para remover objeciones al final, no al inicio.

| Tarea que cruza dominios (ej. una propuesta de cliente que incluye estrategia SEO + ads + script de seguimiento de ventas) | Las referencias relevantes, combinadas | Usa la tabla de circuitos de arriba como columna vertebral y saca de cada referencia solo la sección aplicable — no cargues las tres completas si solo necesitas una sección de cada una |

No cargues un archivo de referencia completo si la tarea es simple (ej. "dame un ejemplo de headline con loss aversion" puede responderse con la tabla de circuitos de arriba, sin abrir los 300+ líneas de `copywriting-marketing.md`). Reserva la carga completa de las referencias para tareas donde necesitas la estructura completa (una secuencia de 5 emails, un script de cold call de 5 pasos, una auditoría de checklist por canal).

El usuario es desarrollador full-stack y especialista en marketing digital freelance en LATAM (Colombia), que suele:
- Escribir propuestas comerciales para clientes (ej. servicios espirituales, ecoturismo, salud) — aplicar con especial cuidado el framing ético en nichos regulados o sensibles.
- Gestionar Meta Ads y Google Ads para clientes LATAM.
- Construir un ecosistema de venta de productos digitales de bajo costo vía WhatsApp con pagos manuales (Nequi/Bre-B), con compromiso explícito de copy no engañoso.

Cuando el contexto de la conversación sea uno de estos, prioriza ejemplos y tono en español, y aplica los guardrails éticos sin que se lo tengas que recordar cada vez.
